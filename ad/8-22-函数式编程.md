# 函数式编程

1.函数式一等公民。
是函数与其他数据类型一样
函数式编程里变量不能被修改，所有的变量只能被赋值一次
2.函数只用表达式，不用语句 （if）
3.没有副作用
4.不修改状态

## 纯函数

对于形同的输入，永远会得到相同的输出，而且没有任何可观察的副作用，也不依赖外部环境的状态

不仅可以有效降低系统的复杂度，还有很多很棒的特性，比如可缓存性。

## 柯里化

传递给函数的一部分参数来调用他，让他返回一个函数去处理剩下的参数。

    var checkage = min => age => age > min;
    var result = checkage(18);
    result(20)

是一种“预加载”函数的方法
